=================================
Changes from Version 1.9.1
=================================



This section addresses the difference between SimPy versions
2.0 and version 1.9.1 in terms of changes and additions.
  


Additions
============

- New (object oriented) API.
- Documentation on running SimPy in parallel processes on multiple CPUs, 
  cores or computerswith Parallel Python

Changes
