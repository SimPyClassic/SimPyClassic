This document addresses the difference between SimPy versions 1.9 and version 1.9.1 in terms of changes and additions.

Repairs

This bug-fix release cures the following bugs:

  • Excessive production of circular garbage, due to a circular reference between Process instances and event notices. This led to large memory requirements.
  • Runtime error for preempts of proceeses holding multiple Resource objects.

Additions

  • A Short Manual, describing only the basic facilities of SimPy.
